게시글 등록기

여러 이미지와 프롬프트를 입력한 뒤 버튼 한 번으로 사이트 폴더에 저장합니다.

사이트 미리보기 ↗
사용 순서

① 사이트 폴더 선택 → ② 내용 입력 → ③ 이미지 추가 → ④ 게시글 저장 → ⑤ GitHub Desktop에서 Commit & Push

1. 사이트 폴더 연결

아직 폴더를 선택하지 않았습니다.

2. 게시글 정보

게시글 전체에 공통으로 표시될 내용입니다.

3. 이미지와 장면별 프롬프트

한 장만 넣으면 일반 게시글, 여러 장을 넣으면 캐러셀 게시글이 됩니다.

이미지를 여기로 끌어오거나 클릭하세요. 여러 장을 한 번에 선택할 수 있습니다.

아직 선택한 이미지가 없습니다.

4. 저장

이미지는 자동으로 이름을 정리해 images 폴더에 넣고, prompts.js도 갱신합니다.

직접 저장 버튼이 작동하지 않을 때

Chrome 또는 Edge에서 이 파일을 여세요. 그래도 폴더 선택 기능이 없으면 ‘수동 저장 파일 받기’를 눌러 내려받은 prompts.js는 사이트 최상단에, 이미지 파일은 images 폴더에 넣으세요.
